# Overview

Project Name is a modern graphical interface for **FFmpeg** that focuses on usability while exposing many of FFmpeg's powerful features.

Instead of remembering long command-line arguments, you can configure everything through an intuitive interface and instantly preview the generated FFmpeg command.

This project was built using **Python** and **PyQt6**.

---

# Features

## Media

- 🎬 Video preview
- 🎵 Audio support
- 📂 Drag & Drop file loading
- 📋 Media information using FFprobe

## Editing

- ✂ Interactive timeline
- 🎯 In / Out trimming
- ⏩ Playback controls
- 🔊 Volume control

## Encoding

- H.264
- H.265 / HEVC
- AV1 (SVT-AV1)
- AV1 (libaom)
- VP9
- Copy streams without re-encoding

## Video Options

- CRF encoding
- Bitrate encoding
- Resolution scaling
- FPS conversion
- Lanczos scaling
- Custom FFmpeg arguments

## Audio Options

- AAC
- MP3
- Opus
- FLAC
- Vorbis
- Audio copy
- Disable audio

## Output

- MP4
- MKV
- WebM
- AVI
- MOV
- GIF
- MP3
- FLAC
- OGG
- WAV

## Additional Features

- ⚡ Live FFmpeg command generation
- 📊 Real-time encoding progress
- 🚀 Encoding speed monitor
- ⏱ ETA estimation
- 📈 Bitrate monitoring
- 📦 Encoding queue
- 🎛 Built-in encoding presets
